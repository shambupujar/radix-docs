# Ecosystem Tools & Libraries

Here you’ll find a list of external dapps, libraries and other tools from across the Radix ecosystem that you may find useful.

:::note[These are unvetted]
Inclusion in this list does not imply endorsement or make any claims of correctness or stability. Please look at the individual service for more details, and judge for yourself if it fits your needs.

If you are are responsible for providing an tool service and would like it to be included here, please get in touch on Discord or at hello@radixdlt.com.
:::



## Analytics, Dashboards and Explorers

- <a href="https://dashboard.radixdlt.com/" target="`_blank`">Official Dashboard</a> (<a href="https://stokenet-dashboard.radixdlt.com/" target="`_blank`">Stokenet</a>) - The official Radix dashboard and explorer.
- <a href="https://www.radxplorer.com/" target="`_blank`">RadXplorer</a>: A community explorer, covering transactions, accounts, assets, validators.
- <a href="https://radixcharts.com/" target="`_blank`">RadixCharts</a>: Analytics platform providing statistics, rankings and charts for the Radix DLT ecosystem.
- <a href="https://www.radixscan.io/" target="`_blank`">RadixScan</a>: A community explorer, covering transactions, accounts, validators, assets and .xrd domains.
- <a href="https://validators.stakesafe.net/" target="`_blank`">StakeSafe Validator Dashboard</a> (<a href="https://stokenet.stakesafe.net/" target="`_blank`">Stokenet</a>): A detailed overview of the validator set.
- <a href="https://www.radixlist.com/" target="`_blank`">Radix List</a>: A listing of ecosystem projects on Radix.
- [Node Manager](https://nodemanager.addix.meme/) by ADDIX - Track XRD flow in/out of nodes at a glance, Get alerts for sudden fee changes, Manage stakes with ease, Node owners can now pull shares directly from the UI

## APIs and Oracles

- <a href="https://radixapi.net/" target="`_blank`">Radix API</a>: Provides additional endpoints and functionality to the official Gateway API to make it easier to obtain specific data.
- <a href="https://docs.astrolescent.com/astrolescent-docs/decentralized-exchange-aggregator/integrations/api" target="`_blank`">Astrolescent API</a>: Find tokens, prices and swap.
- <a href="https://supraoracles.com/" target="`_blank`">Supra Oracles</a> (Coming soon): Offers “push” and “pull” oracle services to deliver precise and reliable price data for more than 100 of the most popular crypto asset pairs.
- <a href="https://religantoracles.com/" target="`_blank`">Religant Oracles</a>: The Religant Oracle protocol enables a decentralized feed of price data.

## Custody

- <a href="https://copper.co/products/digital-asset-custody" target="`_blank`">Copper</a>: Copper provides custody of an impressive and expanding range of digital assets, all insured and held with the highest security standards.
- <a href="https://www.bitpanda.com/en/bitpanda-spotlight" target="`_blank`">Bitpanda</a>: Bitpanda has over 4 million users across 40 countries, opening Radix up to more avenues for adoption within the Web3 community. Bitpanda’s BEST (Bitpanda Ecosystem Token) holders have additional opportunities thanks to this integration. BEST holders are eligible to subscribe to \$XRD and receive giveaway tokens.
- <a href="https://learn.radixdlt.com/article/using-ledger-nano-s-s-plus-or-x-with-the-radix-wallet" target="`_blank`">Ledger</a>: The Radix Wallet can connect to a Ledger Nano S, S Plus, or X hardware wallet and create a special hardware wallet account. This account is protected by the Ledger Nano – even if the Wallet or the PC it is running on is untrusted or compromised in some way. For this account, all transactions must be approved and signed on the Ledger device while connected to a PC that is linked to the Radix Wallet via the Radix Connector browser extension.
- <a href="https://www.getarculus.com/" target="`_blank`">Arculus card</a> (soon): This integration will bring two exciting benefits to the Radix community: a highly convenient way of adding hardware-based security to the Radix Wallet’s coming multi-factor authentication (MFA) account control and an expansion of the reach of the XRD token with a new multi-network mobile wallet app option from Arculus.

## RPC

- <a href="https://www.grove.city" target="`_blank`">Grove</a>: Grove is a pioneer of decentralized RPC infrastructure who has integrated the POKT Network with Radix, adding Radix to their extensive list of supported protocols. With this integration, developers can focus more on building their applications without the hassle of maintaining their own nodes or RPC infrastructure.
- <a href="https://nownodes.io/radix" target="`_blank`">NowNodes</a>: NOWNodes is a Blockchain-as-a-Service solution that provides access to Full Nodes, Explorers and Blockbooks via API. 65+ blockchain networks supported. Choose the node that you need, use available methods and connect.

## On-off ramp

- <a href="https://alchemypay.org" target="`_blank`">Alchemy Pay</a>: Alchemy Pay supports payments in 173 countries and accepts over 50 Fiat currencies via 300+ payment channels. Users can effortlessly convert fiat currencies to XRD and vice versa.
- [Switcher](https://switchere.com/): Switchere supports any credit, prepaid, or debit card issued by VISA, Mastercard, or Maestro.

## Instapass and Instabridge

<a href="https://www.instabridge.io/" target="`_blank`">Instabridge</a> is the first option eXRD token holders have for converting eXRD to XRD for use on the Radix Public Network. If you would like to use Instabridge to convert your eXRD to XRD, you will first need to ensure you have a verified <a href="http://instapass.io/" target="`_blank`">Instapass</a> account. A guide for connecting your Ethereum (eXRD) or Radix (XRD) wallets to your Instapass account can be found <a href="https://www.instapass.io/" target="`_blank`">here</a>.

## Maya

- <a href="https://www.mayaprotocol.com/" target="`_blank`">Maya Protocol</a>: Maya Protocol is a platform specializing in decentralized liquidity and enabling seamless swaps across various blockchains. Once the integration is complete, users of Maya and the Radix Network will be able to trustlessly exchange assets cross-chain without compromising on security.
- <a href="https://www.thorswap.finance/" target="`_blank`">THORSwap</a>: Thorswap enables anyone to interact with liquidity across 15 blockchains, 3 cross-chain protocols and 11 Decentralized Exchanges/ Aggregators. Perform cross-chain swaps between 5,500+ assets (including Native Bitcoin) in a single permissionless, non-custodial transaction.

## Scrypto Libraries

- <a href="https://github.com/ociswap/scrypto-avltree" target="`_blank`">scrypto-avltree</a> by Ociswap: A sophisticated new scalable data structure as alternative to the Btreemap that can be used to house an order book exchange or concentrated liquidity positions.
- <a href="https://github.com/ociswap/scrypto-math" target="`_blank`">scrypto-math</a> by Ociswap: Brings advanced math operations such as exponential, logarithm, and power functions to any Scrypto blueprint that loads the scrypto-math module as a dependency.
- <a href="https://github.com/ociswap/radix-event-stream/tree/main" target="`_blank`">radix-event-stream</a> by Ociswap: An extensible Rust library to help you identify and process events from the Radix ledger.
- <a href="https://github.com/BeakerTools/scrypto-toolkit" target="`_blank`">Scrypto Toolkit</a> by Avincigu : This library aims at providing open source tools for Scrypto developers including Test Engine and Maths library

## Tools by the Community

- **Instruct Manifest Builder** (<a href="https://instruct.radixbillboard.com/" target="`_blank`">Mainnet</a> \| <a href="https://instruct-stokenet.radixbillboard.com/" target="`_blank`">Stokenet</a> \| <a href="https://github.com/radixbillboard/instruct-mainnet-site" target="`_blank`">Code</a>) by Radix Billboard: It allows users to easily and flexibly create custom instructions to be submitted to the network. Not only does it support relevant manifest instructions, it also supports arbitrary Scrypto blueprint schemas.
- **Manifest Checker** ([Mainnet](https://ilikeitstable.com/manifest-checker) \| [Stokenet](https://ilikeitstable.com/manifest-checker-stokenet)) by Stabilis: It validates your manifest and runs a preview, allowing easy debugging of issues with user-build manifests.
- **Xidar Validator Manager** (<a href="https://my.xidar.io/tools/validator-manager" target="`_blank`">Mainnet</a>) by Xidar: It allows validator runners to take standard maintenance actions
- **Manifest Builder** ([Mainnet](https://tmbuilder.stakingcoins.eu/) \| <a href="https://github.com/nemster/tmbuilder" target="`_blank`">Code</a>) by Marco Michelino \| Staking Coins: It features a variety of functionalities, including adding and withdrawing liquidity from Ociswap, depositing and retrieving LSUs from CaviarNine, redeeming WEFT, and facilitating swaps on Ociswap, DefiPlaza, AlphaDex, and RadixPlanet. It is important to note that this tool is intended for educational purposes and should be used with caution. Test it on Mainnet.
- **Desktop Tool** (<a href="https://github.com/atlantis-l/Radix-Desktop-Tool" target="`_blank`">Code</a>) by Hold: The Radix Desktop Tool includes multiple functionalities, such as multiple-to-multiple token transfers, single-to-multiple token transfers, multiple-to-single token transfers, manifest execution, package deployment, token creation, account generation, asset checking, history reviewing, entity verification, format conversion, address QR code generation, and XRD faucet in Stokenet.
- **ERC-404 using Scrypto** (<a href="https://github.com/aus87/ice_rrc404v1" target="`_blank`">Code</a>) by Austin.XRD: ERC-404 is a token standard that combines the features of fungible (ERC-20) and non-fungible (ERC-721) tokens into a hybrid form that enables both unique and divisible asset representations on the blockchain. Inspired by this innovative yet complex token standard, Austin.XRD set out to simplify this using Radix Native Assets, this new token standard on Radix is called RRC-404. Check out the github repository.
